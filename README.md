# Vařečkovi Wedding Website

This repository contains the source code for a simple wedding website.

🌐 Live website: http://varecky.eu/

## About

This project is a small wedding website created to share basic information about the wedding.
It is a simple static website built using basic web technologies.

The goal of the project was to create a clean and easy-to-navigate page where visitors can find important details related to the wedding.

## Technologies Used

* HTML
* CSS
* JavaScript

## Running the Project

1. Clone the repository:

```
git clone https://github.com/R3sll/vareckoviJJ.git
```

2. Open the project folder.

3. Open `index.html` in your web browser.

No installation or build tools are required.
